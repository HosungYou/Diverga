# 최신 연구 레이더 (Research Radar)

**Agent ID**: 08
**Category**: B - 문헌 및 증거
**Icon**: 🌐

## 개요

특정 주제에 대한 신규 출판물을 실시간으로 모니터링하고 연구 트렌드를 분석합니다.
프리프린트, 학술지 출판, 컨퍼런스 발표 등 다양한 출처를 추적합니다.

## 사용 시점

- 연구 주제의 최신 동향을 파악할 때
- 경쟁 연구팀의 출판물을 추적할 때
- 핵심 논문의 인용 논문을 모니터링할 때
- 새로운 연구 기회를 탐색할 때

## 핵심 기능

1. **자동 알림 설정**
   - 키워드 기반 알림
   - 저자 기반 알림
   - 저널 기반 알림
   - 인용 알림

2. **프리프린트 모니터링**
   - arXiv, SSRN, OSF
   - bioRxiv, medRxiv
   - 분야별 프리프린트 서버

3. **트렌드 분석**
   - 신흥 연구 주제 식별
   - 연구 방법론 트렌드
   - 인용 패턴 분석

4. **요약 보고서 생성**
   - 주간/월간 업데이트
   - 핵심 논문 하이라이트
   - 관련성 점수

## 모니터링 플랫폼

| 플랫폼 | 유형 | 특징 | 알림 방법 |
|--------|------|------|----------|
| Google Scholar | 종합 | 광범위한 커버리지 | 이메일 |
| Semantic Scholar | 종합 | AI 기반 추천 | API |
| arXiv | 프리프린트 | CS, 물리, 수학 | RSS, API |
| SSRN | 프리프린트 | 사회과학, 경영 | RSS |
| OSF Preprints | 프리프린트 | 다분야 | API |
| PubMed | 의학 | MEDLINE 색인 | RSS, 이메일 |

## 입력 요구사항

```yaml
필수:
  - 추적 키워드: "주제 관련 키워드"
  - 관심 분야: "학문 영역"

선택:
  - 관심 저자: "추적할 연구자"
  - 핵심 논문: "인용 추적할 논문"
  - 기간: "모니터링 기간"
  - 언어: "선호 언어"
```

## 프롬프트 템플릿

```
당신은 학술 문헌 모니터링 전문가입니다.

다음 주제에 대한 최신 연구 동향을 파악해주세요:

[추적 키워드]: {keywords}
[관심 저자]: {authors}
[기간]: {time_period}

수행할 작업:
1. 최신 출판물 탐색
   - 지정 기간 내 출판된 논문 검색
   - 프리프린트 포함
   - 학회 발표 포함

2. 각 논문에 대해:
   - 제목, 저자, 출처, 날짜
   - 초록 요약 (3-4문장)
   - 핵심 발견/기여
   - 현재 연구와의 관련성 (1-5점)

3. 트렌드 분석
   - 주요 연구 방향
   - 새롭게 부상하는 개념/방법
   - 연구 공백

4. 우선순위 논문 추천
   - 반드시 읽어야 할 논문 (상위 5개)
   - 각 논문의 중요성 근거
```

## 알림 설정 가이드

### Google Scholar Alerts
```
1. scholar.google.com 접속
2. 검색 실행
3. "Create alert" 클릭
4. 이메일 설정
```

### Semantic Scholar
```
1. semanticscholar.org 접속
2. 저자/주제 검색
3. "Create Alert" 또는 "Follow"
4. 알림 주기 설정
```

### arXiv
```
1. arxiv.org 접속
2. 관심 카테고리 선택
3. RSS 피드 구독 또는
4. arXiv API 활용
```

### PubMed
```
1. pubmed.ncbi.nlm.nih.gov 접속
2. 검색 실행
3. "Create alert" 클릭
4. 이메일 주기 설정
```

| 점수 | 기준 |
|------|------|
| ⭐⭐⭐⭐⭐ | 직접적 관련, 즉시 인용 필요 |
| ⭐⭐⭐⭐ | 높은 관련성, 방법론/이론 참고 |
| ⭐⭐⭐ | 중간 관련성, 배경으로 참고 |
