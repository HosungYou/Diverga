## 1. 에이전트 계약 검증 시스템 (Agent Contract Validation System)

### 1.1 목적

SKILL.md 파일이 Research Coordinator의 계약 스키마를 준수하는지 자동으로 검증하는 시스템입니다.

### 1.2 파일 위치

```
scripts/validate_agents.py
```

### 1.3 주요 기능

| 기능 | 설명 |
|------|------|
| **YAML 프론트매터 파싱** | 외부 의존성 없이 YAML 파싱 (PyYAML 불필요) |
| **필수 필드 검증** | name, version, description, upgrade_level, v3_integration |
| **시맨틱 버전 검사** | X.Y.Z 형식 권장 (경고 발생) |
| **업그레이드 레벨 검증** | FULL, ENHANCED, LIGHT 유효성 검사 |
| **V3 통합 요구사항** | FULL 에이전트: dynamic_t_score=true, 5개 창의성 모듈, 6개 이상 체크포인트 |

### 1.4 사용법

```bash
# 모든 에이전트 검증
python scripts/validate_agents.py

# 특정 에이전트 검증
python scripts/validate_agents.py --agent 02

# 상세 출력
python scripts/validate_agents.py --verbose

# JSON 출력 (CI/CD용)
python scripts/validate_agents.py --json
```

### 1.5 검증 결과

```
============================================================
 Research Coordinator Agent Validation Results
============================================================

✅ 01-research-question-refiner: PASS
✅ 02-theoretical-framework-architect: PASS
✅ 03-devils-advocate: PASS
...
✅ 21-conceptual-framework-visualizer: PASS

------------------------------------------------------------
Summary: 21/21 agents passed validation
✅ All agents passed validation!
```

### 1.6 검증 규칙

#### FULL 에이전트 요구사항
- `dynamic_t_score: true` 필수
- 5개 창의성 모듈 권장
- 6개 이상 체크포인트 권장
- VS 방법론 문서화 필요

#### ENHANCED 에이전트 요구사항
- 3개 이상 창의성 모듈 권장
- VS 방법론 문서화 필요

#### LIGHT 에이전트 요구사항
- 기본 프론트매터 필드만 필요

## 2. 크로스 플랫폼 CLI 도구 (Cross-Platform CLI Tool)

### 2.1 목적

연구자들이 Research Coordinator 에이전트를 쉽게 관리할 수 있는 명령줄 도구입니다.

### 2.2 파일 위치

```
scripts/rc        # Bash 버전 (macOS/Linux)
scripts/rc.py     # Python 버전 (모든 OS)
```

### 2.3 사용 가능한 명령어

| 명령어 | 별칭 | 설명 |
|--------|------|------|
| `help` | `-h`, `--help` | 도움말 표시 |
| `status` | - | 설치 상태 확인 |
| `list` | `ls` | 모든 에이전트 목록 |
| `validate` | `check` | 에이전트 계약 검증 |
| `info` | `show` | 에이전트 상세 정보 |
| `doctor` | `diagnose` | 설치 문제 진단 |
| `update` | `upgrade` | 최신 버전으로 업데이트 (Bash만) |
| `uninstall` | `remove` | 설치 제거 (Bash만) |
| `version` | `-v`, `--version` | 버전 정보 |

### 2.4 사용 예시

```bash
# Bash 버전 (macOS/Linux)
./scripts/rc list
./scripts/rc info 02
./scripts/rc validate

# Python 버전 (모든 OS)
python scripts/rc.py list
python scripts/rc.py info 02
python scripts/rc.py validate
```

## 3. Windows 지원 (Windows Support)

### 3.1 PowerShell 설치 스크립트

**파일 위치**: `scripts/install.ps1`

### 3.2 사용법

```powershell
# 심볼릭 링크로 설치 (관리자 권한 필요)
.\scripts\install.ps1

# 파일 복사로 설치 (관리자 권한 불필요)
.\scripts\install.ps1 -Copy

# 설치 제거
.\scripts\install.ps1 -Uninstall

.\scripts\install.ps1 -Help
```

- 관리자 권한 자동 감지

- 한국어/영어 이중 언어 출력
- 컬러 터미널 지원
